# Trainer Grammar

Definitions for common terms within this application context.

## Domain
### Activity
A capture of a point in time event, with additional details including its [Type](#type).

### Activity Type
See [Type](#type).

### Type
A definition of a certain kind of [activity](#activity).

## User interface

### Main Screen
The first interactive UI shown to the user when starting up the application.

### Activity Screen
The UI that allows the user to create / edit an [activity](#activity).

### Activity Type Screen
The UI that allows the user to create / edit a [activity type](#activity-type).
